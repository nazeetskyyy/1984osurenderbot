# osu! Replay Render Bot

Telegram бот для рендера osu! реплеев через сервис o!rdr.

## 📋 Требования

- Python 3.8 или выше
- PyCharm (или любая другая IDE)
- Токен Telegram бота (получить у @BotFather)
- API ключ o!rdr (зарегистрироваться на ordr.issou.best)
